# Облік дерев.
#### Даний модуль має такі можливості:
* Додати тип дерева,
* Додати сорт дерева,
* Додати дерево,
* Пошук дерева,
* Редагування дерева,
* Редагування сорта,
* Редагування типу

Щоб підключити даний модуль вам варто відкрити файл config.pl який знаходиться за шляхом /us/abills/libexec/config.pm

В даному файлі знаходимо масив @MODULES і записуємо назву модуля.
# Приклад:
###### our @MODULES = (
######   'MyTmp',
######   'Tree',
###### );

Сам модуль потрібно занести в папку за шляхом /usr/abills/Abills/modules/[Назва модуля]
