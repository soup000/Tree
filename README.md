# Облік дерев.
#### Даний модуль має такі можливості:
* Додати тип дерева,
* Додати сорт дерева,
* Додати дерево,
* Пошук дерева,
* Редагування дерева,
* Редагування сорта,
* Редагування типу

#### Установка:
Створити таблицю в базі:
```
# mysql --default-character-set=utf8 -D abills < db/Tree.sql
```
В файлі **config.pl**:
```
@MODULES = (
             'Tree'
           );
```
